Book Price Finder

This is a test application which uses a local Dino Broker to invoke services. It will confirm that Dino is working and demonstrates how to use the Dino Broker. The source code is found in the "src" folder. This example is quite limited in that it makes use of only a single service. It does however use a real service provided by an external entity.

To Run

Easy way

Run bookFinder.jar in the BookPriceFinder folder, for example by clicking or double clicking on it. This should work in many cases. If it doesn't then use one of the methods below.

Java command line

At the console / command prompt, navigate to the BookPriceFinder directory and then execute "java -jar bookFinder.jar".

Windows

Execute "run.bat" in the BookPriceFinder folder.

Ant

At the console / command prompt, navigate to the BookPriceFinder directory and then execute "ant run".

To Recompile

You will need the latest version of Ant.

Execute "ant" to compile and create bookFinder.jar. Execute "ant run" to run.

Short Overview of the Code

There are four classes. The most important is the BookPriceFinder class as this contains the Dino Broker code. The "initBroker" method sets up the Dino Broker and registers the ReqDoc reqdoc.xml. The other important method is "findBookPrice(String bookISBN)" which invokes the price finder service when the user clicks on the "search" button.

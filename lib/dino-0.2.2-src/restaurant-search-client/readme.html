Restaurant Search

This is a more complex example.

To Run

Easy way

Run restaurantSearch.jar in the RestaurantSearch folder, for example by clicking or double clicking on it. This should work in many cases. If it doesn't then use one of the methods below.

Java command line

At the console / command prompt, navigate to the BookPriceFinder directory and then execute "java -jar restaurantSearch.jar".

Windows

Execute "run.bat" in the BookPriceFinder folder.

Ant

At the console / command prompt, navigate to the BookPriceFinder directory and then execute "ant run".

To Recompile

You will need the latest version of Ant.

Execute "ant" to compile and create restaurantSearch.jar. Execute "ant run" to run.
